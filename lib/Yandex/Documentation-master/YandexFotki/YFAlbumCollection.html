  • 
				string
				$order: Порядок отображения элементов выдачи. Допустимые значения: updated (по умолчанию) - по времени последнего изменения, от новых к старым; rupdated - по времени последнего изменения, от старых к новым; published - по времени загрузки (для фотографии) или создания (для альбома), от новых к старым; rpublished - по времени загрузки (для фотографии) или создания (для альбома), от старых к новым;			
    • 
					
  • 
				string
				$offset_time: Время создания ресурса в формате UTC с точностью до секунды.			
    • 
					
  • 
				string
				$offset_id: Численный идентификатор ресурса на Яндекс.Фотках.			
    • 
					
  • 
				int
				$limit: Количество элементов на странице выдачи.			
    •
